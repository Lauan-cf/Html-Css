Parágrafos e quebras de linha

Você pode escrever um parágrafo de qualquer jeito basta estar entre um par de <p> e </p>

Se precisar quebrar o
texto em algum lugar expecífico use a tag <br>

Se quiser usar o sinal de menor ou maior sem criar tag
pode fazer assim <p>
( trascrevendo para quem só tá no site )
para o sinal com o maior pra direita → usar o "&l" e após isso colocar um "t;"
para o sinal com o maior pra direita → usar o "&g" e após isso colocar um "t;"

Caracteres especiais

Começando com os símbolos:

• ® (marca registrada)
• © (copyright)
• ™
• €
• ¥
• ¢
• ℜ
• δ
• ↑ ou ↑
• ⇐
(etc...)

além dos acima há uma ampla gama de símbolos
os quais compõem todos os caracteres aceitos no seu computador

Também há os emojis, os quais são feitos de uma forma um pouco similar porém tendo a obrigatoriedade
de ter um "&#x" para dizer que o valor é hexadecimal:

Exemplos: ⭐, 🔥
