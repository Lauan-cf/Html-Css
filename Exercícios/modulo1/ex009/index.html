Listas não Ordenadas

Como exemplo há uma lista de compras:

  • Arroz

  • Macarrão
  • ...

Listas de Definição

Funcionam como um dicionário, assim ligando um termo a sua definição, sendo usado margens como forma de representação. Ex.:

Html
Liguagem de marcação para a criação do conteúdo de um site.
CSS
Linguagem de marcação para o design de um site.
JavaScript
Linguagem de programação para a criação de interatividade de um site
